# 席替えアプリ

クラスの座席表を自動生成するWebアプリです。ブラウザだけで動作し、データは端末内に保存されます。

## 使い方

### 1. 生徒を登録する（生徒管理タブ）

- 手動で一人ずつ追加するか、CSV/TSVを貼り付けて一括追加できます。
- CSV形式: `名前, 性別, 視力, 身長, メモ`

### 2. 座席レイアウトを設定する（座席設定タブ）

- 縦・横の数を設定します。
- 座席をクリックして「利用不可」や「固定」を設定できます。

### 3. 席替えを生成する（生成・履歴タブ）

- 「席替えを生成」ボタンを押すと最適な配置を生成します。
- 過去の履歴を参照し、同じ席・同じ隣同士を避けます。

## データの保存・共有

- データは自動的にブラウザのローカルストレージに保存されます。
- 「情報を保存」でJSONファイルとして書き出せます。
- 「情報を読込」でJSONファイルを読み込めます。

## 動作環境

- Chrome / Safari / Firefox（最新版）
- スマートフォン（iPhone Safari）対応
